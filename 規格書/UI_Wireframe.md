### 導航邏輯

| 起點 | 終點 | 觸發方式 |
|------|------|---------|
| 首頁 | 個股分析 | 點擊股票代號 / 搜尋框輸入 |
| 個股分析 | 產業分析 | 點擊「查看產業詳情」按鈕 |
| 個股分析 | 催化劑 | 點擊「查看催化劑」按鈕 |
| 多股比較 | 個股分析 | 點擊任一股票卡片 |
| 復盤系統 | 個股分析 | 點擊交易紀錄中的股票 |

## 2. 首頁儀表板

### 元件清單

| 元件 | 類型 | 說明 |
|------|------|------|
| 歡迎訊息 | Text | 「歡迎回來，[使用者名稱]」 |
| 快速搜尋 | SearchBar | 輸入股號或公司名稱 |
| 最近查詢 | List | 最近 10 檔查詢紀錄 |
| 市場概覽 | Card | 加權指數、成交量、漲跌家數 |
| 關注股票 | Table | 使用者追蹤的股票清單 |
| 系統通知 | Banner | 資料更新狀態、系統公告 |

### 互動邏輯

- **搜尋框**：輸入股號後按 Enter，跳轉至個股分析頁
- **最近查詢**：點擊任一項目，跳轉至該股票分析頁
- **關注股票**：點擊「+」新增追蹤，點擊股票代號跳轉分析頁

## 3. 個股分析頁

### 元件清單

| 元件 | 類型 | 說明 |
|------|------|------|
| 股票代號 | Text | 顯示目前分析的股票 |
| 公司名稱 | Text | 顯示公司全名 |
| 產業分類 | Tag | 顯示所屬產業 |
| 五大維度雷達圖 | Chart | 雷達圖顯示五維度評分 |
| 維度詳情卡片 | Card × 5 | 每個維度的詳細評分與說明 |
| 目標 PE 定價 | Card | Tier 等級、目標 PE 區間 |
| 追價安全圈 | Card | 雙軌制結果（安全邊際法 + DCF） |
| 目前股價 | Text | 顯示最新收盤價 |
| 股價位置指示器 | Gauge | 顯示股價在安全圈的位置 |
| 操作按鈕 | Button | 「查看產業詳情」「查看催化劑」「匯出 HTML」 |

### 互動邏輯

- **維度詳情卡片**：點擊展開詳細說明，顯示系統建議分數與手動調整記錄
- **追價安全圈**：雙軌制結果並列顯示，附帶詳細計算說明
- **匯出 HTML**：點擊後產生分析報告 HTML 檔案

### 資料流向

```
輸入股號
  ↓
呼叫 Layer 2 核心引擎
  ↓
取得五大維度評分
  ↓
計算目標 PE + 安全圈
  ↓
顯示結果
```

## 附錄：元件規格

### 圖表規格

| 圖表類型 | 用途 | 規格 |
|---------|------|------|
| 雷達圖 | 五大維度評分 | 5 軸，每軸 0-20，支援多線疊加 |
| 圓餅圖 | 資產配置 | 3 區塊（核心/衛星/避險） |
| 折線圖 | 走勢對比 | X 軸時間，Y 軸股價，支援多線 |
| 長條圖 | 同業比較 | 群組長條圖，對比多家數據 |
| 儀表圖 | 股價位置 | 0-100% 區間，標示便宜/合理/昂貴 |

### 表格規格

| 表格類型 | 用途 | 規格 |
|---------|------|------|
| 資料表格 | 交易紀錄、同業比較 | 支援排序、篩選、分頁 |
| 卡片表格 | 關注股票 | 每列顯示股票卡片，可點擊跳轉 |
